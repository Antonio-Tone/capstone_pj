Home

HTTP Methods Path Description
GET /(Home page) Takes you to the home page

Users Table

HTTP Methods Path Description
GET /users displays a list of users
GET /user/:id displays a specific user based on id
POST /register Creates a new user
PATCH/PUT /user/:id Updates an existing user or adds a user
DELETE /user/:id Deletes a user

Vehicles Table

HTTP Methods Path Description
GET /Vehicles Displays a list of vehicles
GET /Vehicle/:id Displays a single vehicle by id
POST /Vehicle Adds a new vehicle
PUT/PATCH /Vehicle/:id Update a vehicles details
DELETE /Vehicle/:id deletes a vehicle from the database

Orders Table

HTTP Methods Path Description
GET /Orders Displays a list of orders
GET /Order/:id Displays a single order by id
POST /Order Adds a new order
PUT/PATCH /Order/:id Update an order details
DELETE /Order/:id deletes a order from the database
